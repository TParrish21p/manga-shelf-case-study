# Release Process

Manga Shelf was built and released as an Android app through Google Play Console.

## Release steps

- local Flutter testing
- physical Android device testing
- signed Android App Bundle builds
- closed testing
- tester feedback updates
- Play Console production submission

## Release considerations

The app includes:

- privacy policy
- terms of service
- data deletion page
- account deletion page
- Data safety disclosures
- account deletion support inside the app
